Installation¶

DFFML currently only supports Python 3.7. If your distribution’s package manager doesn’t provide Python 3.7, pyenv is another good way to install it.

DFFML is available via pip.

python3.7 -m pip install dffml

If you want to stay on the bleeding edge of bug fixes, etc. Install via git.

python3.7 -m pip install git+https://github.com/intel/dffml

You can also install the Features for Git Version Control, and Models for Tensorflow Library all at once.

python3.7 -m pip install -U dffml[git,tensorflow]
